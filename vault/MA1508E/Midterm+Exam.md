# MA1508E - Midterm Exam Overview

## Overview
This note summarizes the key components and structure of the Midterm Exam for MA1508E - Linear Algebra for Engineering at the National University of Singapore for the AY2023/2024 Semester 2. The exam consists of multiple-choice questions (MCQs) and short answer questions, focusing on linear systems, matrix operations, and vector spaces. Students are required to demonstrate their understanding of these concepts through problem-solving and analytical reasoning.

## Key Concepts & Definitions
- **Linear System**: A collection of linear equations involving the same set of variables. [[Linear Algebra]]
- **Matrix Product**: The result of multiplying two matrices, which is defined when the number of columns in the first matrix equals the number of rows in the second. [[Matrix Operations]]
- **Determinant**: A scalar value that is a function of a square matrix, providing important properties such as whether the matrix is invertible. [[Determinants]]
- **Subspace**: A set of vectors that forms a vector space under vector addition and scalar multiplication. [[Vector Spaces]]

## Detailed Technical Breakdown

### Exam Structure
- **Duration**: 1 hour 30 minutes
- **Format**:
  - 15 Multiple Choice Questions (MCQs)
  - 3 Short Answer Questions
- **Submission**:
  - MCQs submitted via Examplify
  - Short answers written in provided answer booklet and submitted as a PDF

### Sample Questions Breakdown
1. **Linear System Example**:
   - Given a system of equations, determine the value of \( x \) when \( z = 3 \).
   - Options: (a) -9, (b) -5, (c) 0, (d) 1, (e) None of the other options.

2. **Quadratic System Analysis**:
   - Analyze the implications of \( z > 1 \) on the values of \( x \) and \( y \).
   - Options: (a) \( x > 0, y < 1 \), (b) \( x > 1, y > 0 \), etc.

3. **Matrix Properties**:
   - Discuss the properties of the product of two matrices \( A \) and \( B \).
   - Options: (a) Not invertible, (b) Invertible, etc.
