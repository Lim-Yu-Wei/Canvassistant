# On-Campus Examplify Student Briefing

## Overview
This note provides a comprehensive guide for students at the National University of Singapore (NUS) on using Examplify for on-campus examinations. It covers essential device requirements, installation steps, registration procedures, and tips for taking practice exams. Additionally, it highlights important features of the Examplify interface and troubleshooting support.

## Key Concepts & Definitions
- **Examplify**: A secure exam software used for administering assessments.
- **NUSNET**: The university's network login credentials required for accessing Examplify.
- **Secure Exam**: An exam mode that restricts access to other applications and internet browsing.
- **Non-Secure Exam**: An exam mode that allows access to other applications during the assessment.

## Detailed Technical Breakdown

### Device Requirements
- **Operating Systems**:
  - **Windows**: Windows 10 (22H2), Windows 11 (21H2, 22H2)
  - **MacOS**: Big Sur, Monterey, Ventura (Intel, M1, M2)
- **Screen Resolution**: Minimum of 1280 x 768
- **Display Scale**: Set to 100%

### Installation Steps
1. **Download Examplify**:
   - Use Chrome or Firefox to visit [ExamSoft NUS](http://www.examsoft.com/nus).
   - Log in with NUSNET credentials.
   - Download and install the application.
   - Create an exclusion in your anti-virus software.

2. **Register for Examplify**:
   - Connect to Wi-Fi.
   - Launch Examplify and enter Institution ID: `nus`.
   - Follow prompts to log in and access the dashboard.

### Taking Practice Exams
- **Institution ID**: NUS APAC
- **Practice Password**: `password1`
- Multiple attempts allowed for practice exams.

### During the Exam
- **Important Notes**:
  - Wi-Fi is not required during the exam.
  - Timer starts when the first question is displayed.
  - Auto-save feature is enabled.

#### Shortcut Keys
| Windows         | Macintosh       |
|------------------|-----------------|
| Ctrl + > : Next Question | ⌘ + > : Next Question |
| Ctrl + < : Prev Question | ⌘ + < : Prev Question |
| Ctrl + H : Hide Exam | ⌘ + H : Hide Exam |
| Ctrl + G : Show Exam Attachment | ⌘ + G : Show Exam Attachment |

> [!warning] **Important**: Avoid using shortcut keys during exams that do not allow returning to previous questions.

## Implementation/Examples

```plaintext
1. To submit your exam:
   - Click on Exam Controls > Submit Exam.
   - Confirm completion and click Submit.
```
