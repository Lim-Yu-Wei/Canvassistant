# MA1508E - Linear Algebra for Engineering: Examination Overview

## Overview
This note summarizes the examination structure and key concepts covered in the MA1508E course at the National University of Singapore for Semester 1 of AY2021/2022. The exam consists of five questions, focusing on matrix theory, linear transformations, and differential equations, with an emphasis on systematic problem-solving using tools like MATLAB. Students are required to demonstrate their understanding of concepts such as [[row echelon form]], [[basis]], and [[eigenvectors]].

## Key Concepts & Definitions
- **Matrix**: A rectangular array of numbers arranged in rows and columns, used to represent linear transformations.
- **Row Echelon Form**: A form of a matrix where all non-zero rows are above any rows of all zeros, and leading coefficients of a non-zero row are to the right of the leading coefficients of the previous rows.
- **Basis**: A set of vectors in a vector space that are linearly independent and span the space.
- **Eigenvector**: A non-zero vector that changes by only a scalar factor when a linear transformation is applied.
- **Transition Matrix**: A square matrix used to describe the transitions of a Markov chain.

## Detailed Technical Breakdown

### Examination Instructions
- **Duration**: 2 hours
- **Format**: Open book, proctored via Zoom
- **Materials Allowed**: MATLAB, scientific or graphing calculators
- **Submission**: Candidates must submit a PDF of their work post-examination.

### Question Breakdown
1. **Matrix Consistency** (18 marks)
   - Conditions for the system \( Ax = b \) to be consistent for all \( b \in \mathbb{R}^4 \).
   - Reduced row echelon form analysis.
   - Basis for the row space and nullspace.

2. **Basis and Orthonormality** (15 marks)
   - Demonstrating that a set \( S \) is a basis for a vector space \( V \).
   - Applying the [[Gram-Schmidt process]] to convert \( S \) to an orthonormal basis.

3. **Markov Chain Analysis** (18 marks)
   - Constructing a transition matrix for patrol routes.
   - Finding state vectors after multiple transitions and diagonalizing the transition matrix.

4. **Eigenvalues and Differential Equations** (18 marks)
   - Identifying complex and real eigenvectors.
   - Solving a system of first-order linear differential equations.

5. **First Order Linear Systems** (11 marks)
   - Solving a specified system of differential equations with initial conditions.
