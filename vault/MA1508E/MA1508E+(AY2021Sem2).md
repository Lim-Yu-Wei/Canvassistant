# MA1508E - Linear Algebra for Engineering (AY2020/2021)

## Overview
This note summarizes the key components of the MA1508E course, focusing on linear algebra concepts relevant to engineering applications. It includes examination instructions, problem-solving strategies, and essential definitions. The course emphasizes the understanding of linear systems, polynomial interpolation, subspaces, matrix operations, and differential equations.

## Key Concepts & Definitions
- **Linear System**: A collection of linear equations involving the same set of variables. 
- **Polynomial Interpolation**: The process of determining a polynomial that passes through a given set of points.
- **Subspace**: A subset of a vector space that is closed under vector addition and scalar multiplication.
- **Matrix Inverse**: A matrix that, when multiplied with the original matrix, yields the identity matrix.
- **Determinants**: A scalar value that is a function of the entries of a square matrix, providing important properties about the matrix.
- **Differential Equations**: Equations that involve derivatives of a function.

## Detailed Technical Breakdown

### Examination Instructions

### Problem Breakdown
1. **Linear Systems**:
   - Conditions for no solution, unique solution, and infinitely many solutions.
   - Example: Given a system of equations, determine the conditions on constants \(a\) and \(b\).

2. **Polynomial Interpolation**:
   - Finding a polynomial of degree \(n\) that fits a set of points.
   - Minimizing the error for polynomial approximations.

3. **Subspaces**:
   - Demonstrating that a set is a subspace by finding a spanning set.
   - Using the Gram-Schmidt process to create an orthonormal basis.

4. **Matrix Operations**:
   - Finding left and right inverses of matrices.
   - Computing determinants and finding bases for intersections of column spaces and nullspaces.

5. **Differential Equations**:
   - Solving systems of linear differential equations with initial conditions.
