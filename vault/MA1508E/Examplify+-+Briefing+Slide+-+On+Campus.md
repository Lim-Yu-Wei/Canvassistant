# On-Campus Examplify Student Briefing

## Overview
This note provides a comprehensive guide for students at the National University of Singapore (NUS) on how to prepare for and take exams using the Examplify software. It covers device requirements, installation steps, registration procedures, and tips for successfully navigating the exam interface. Understanding these guidelines will ensure a smooth examination experience.

## Key Concepts & Definitions
- **Examplify**: A secure exam software used for administering assessments.
- **Minimum System Requirements**: Specifications that devices must meet to run Examplify effectively.
- **Secure Exam**: An exam mode that restricts access to other applications and the internet during the assessment.
- **Non-Secure Exam**: An exam mode that allows access to other applications and the internet.

## Detailed Technical Breakdown

### Device Requirements
- **Operating Systems**:
  - **Windows**: Windows 11 versions 23H2, 24H2, and 25H2.
  - **Mac OS**: Supported versions include Sonoma, Sequoia, and Tahoe.
- **Language Settings**: Set the default language to English (US).
- **Screen Resolution**: Minimum of 1280 x 768 with a display scale of 100%.

### Installation Steps
1. **Download Examplify**:
   - Use Chrome or Firefox to visit [ExamSoft NUS](http://www.examsoft.com/nus).
   - Sign in with your NUS email.
   - Download and install the application.
2. **Register for Examplify**:
   - Connect to Wi-Fi and launch Examplify.
   - Enter "NUS" in the Institution field and select "National University of Singapore-APAC (nus)".
   - Follow prompts to log in and access the exam dashboard.

### Taking Practice Exams
- **Institution ID**: NUS APAC
- **Practice Password**: Examplify101
- Multiple attempts are allowed for practice exams.

### During the Exam
- **Wi-Fi**: Not required during the exam.
- **Timer**: Starts when the first question appears.
- **Auto-Save**: Examplify automatically saves answers.

### Shortcut Keys
| Windows Shortcuts | Macintosh Shortcuts |
|-------------------|---------------------|
| Ctrl + > : Next Question | ⌘ + > : Next Question |
| Ctrl + < : Prev Question | ⌘ + < : Prev Question |
| Ctrl + H : Hide Exam | ⌘ + H : Hide Exam |
| Ctrl + G : Show Exam Attachment | ⌘ + G : Show Exam Attachment |
| Ctrl + N : Show Notices | ⌘ + N : Show Notices |

> [!important]  
> Ensure your device is fully charged and bring your charger to the exam venue.

> [!warning]  
> Do not upgrade your operating system before the exam, especially to a beta version, as it may cause compatibility issues with Examplify.
