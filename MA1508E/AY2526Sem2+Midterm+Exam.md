## Key Concepts
- [[System of Linear Equations]] — A collection of linear equations involving the same set of variables.
- [[Matrix Invertibility]] — A square matrix is invertible if there exists another matrix that, when multiplied, yields the identity matrix.
- [[Determinant]] — A scalar attribute of a square matrix that can indicate if the matrix is invertible.
- [[Row Echelon Form]] — A matrix form used to simplify solving systems of linear equations.
- [[Polynomial Functions]] — Mathematical expressions that involve sum of powers in one or more variables multiplied by coefficients.
