# CG2111A Mid-Term Assessment Overview

The CG2111A Mid-Term Assessment evaluates students' understanding of engineering principles and practices related to microcontroller programming and system design. This assessment consists of twenty questions covering various topics, including [[Timers]], [[Interrupts]], and [[Polling]]. Each question is worth one mark, contributing to 10% of the overall course score. Students must adhere to strict guidelines regarding answer submission and academic integrity.

## Key Concepts & Definitions

- **Inter-Arrival Time (IAT)**: The time interval between bytes arriving at the processor from the disk drive.
- **Polling**: A method where the processor repeatedly checks the status of a device.
- **Interrupts**: Signals that temporarily halt the processor's current operations to execute a specific task.
- **Direct Memory Access (DMA)**: A feature that allows peripherals to communicate with memory without CPU intervention.
- **Timer**: A hardware component that counts clock cycles to measure time intervals or generate events.

## Detailed Technical Breakdown

### Assessment Structure
- **Total Questions**: 20
- **Total Marks**: 20 (10% of overall score)
- **Duration**: 60 minutes
- **Submission Requirements**:
  - Use of an OCR form for answers.
  - No electronic devices except for a standalone calculator.
  - Communication restrictions during the assessment.

### Key Questions Breakdown
1. **Inter-Arrival Time Calculation**:
   - Given a disk drive transfer rate of 150 MBPS and a bus throughput of 120 MBPS, calculate the smallest possible IAT.
   - Options: Approx 4.17 ns, 6.66 ns, 8.33 ns, 16.66 ns.

2. **Polling Cycles Calculation**:
   - Analyze a code snippet to determine the total cycles spent polling the DATA_READY register.

3. **Interrupt Processing Time**:
   - Calculate the time taken to process a single interrupt based on given cycle timings.

4. **Polling vs. Interrupts vs. DMA**:
   - Evaluate statements regarding the efficiency and requirements of each method.

5. **ATMega328P Interrupts**:
   - Assess the truth of statements regarding the interrupt capabilities of the ATMega328P.

### Timers and PWM
- **Timer Configuration**:
  - Understand the settings for Timer 1 and Timer 0, including registers like TCCR1A, TCCR1B, and TIMSK1.
  
- **PWM Duty Cycle Calculation**:
  - Analyze code snippets to determine the duty cycle of PWM signals generated.

## Implementation/Examples

```c
// Example of Timer 0 setup for PWM
#include "Arduino.h"
#include <avr/io.h>
#include <avr/interrupt.h>

void setup() {
    TCNT0 = 0; // Initialize Timer Counter
    TCCR0A = 0b10000001; // Set Timer 0 to Fast PWM mode
    TIMSK0 |= 0b10; // Enable Timer 0 Compare Match Interrupt
    OCR0A = 128; // Set Output Compare Register for 50% duty cycle
    DDRD |= (1 << 6); // Set PD6 as output
    sei(); // Enable global interrupts
}

ISR(TIMER0_COMPA_vect) {
    // ISR code here
}
```
