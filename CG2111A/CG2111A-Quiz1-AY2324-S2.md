# CG2111A Engineering Principles and Practices II - Quiz 1 Overview

## Overview
This note summarizes the key points and technical details from the CG2111A Quiz 1, focusing on the configuration and control of LEDs and switches using the AT328P microcontroller. It covers essential programming concepts, including the use of masks for bit manipulation, PWM signal generation, and the handling of interrupts. The quiz also tests understanding of data transmission principles and the configuration of USART settings.

## Key Concepts & Definitions
- **AT328P**: A microcontroller used in Arduino boards.
- **PWM (Pulse Width Modulation)**: A technique used to control the amount of power delivered to an electronic device.
- **Interrupts**: Mechanisms that allow a microcontroller to respond to events asynchronously.
- **USART (Universal Synchronous/Asynchronous Receiver-Transmitter)**: A hardware communication protocol for serial data transmission.
- **Masking**: The process of using bitwise operations to isolate or manipulate specific bits in a byte.

### LED Configuration
- **Pin to Port Mappings**:
  - Arduino Pin 8 → Port B, Pin 0
  - Arduino Pin 7 → Port D, Pin 7

#### C Function for Masking
```c
unsigned char MASK(unsigned char bit_position) {
    return (0x01 << bit_position);
}
```

#### DDR Configuration Options
- **Correct Configuration**: 
  - `DDRB |= MASK(0);`
  - `DDRD |= MASK(7);`

### Switch Configuration
- **Switch Connection**: Port B Pin 5 (PB5)
- **Correct Configuration**: 
  - `DDRB &= ~MASK(5);`

### LED Control Logic
- **Control Logic**:
```c
if ((PINB & MASK(5)) != 0x00) {
    LED_On();
} else {
    LED_Off();
}
```

### PWM Signal Generation
- **Duty Cycle Calculation**:
  - **Code Snippet**:
```c
#include "Arduino.h"
#include <avr/io.h>
#include <avr/interrupt.h>
unsigned char val = 0x01;
void setup() {
    TCNT0 = 0; 
    TCCR0A = 0b10100001; 
    OCR0A = (val << 7); 
    OCR0B = (val << 6); 
    TCCR0B = 0b00000011; 
    DDRD |= (val << 6) | (val << 5);
}
```
- **Duty Cycle Results**:
  - PD6 = 25%, PD5 = 50%

### Interrupt Handling
- **ISR Definitions**:
```c
ISR(TIMER0_COMPA_vect) {
    // Code for Timer Compare A
}
ISR(TIMER0_COMPB_vect) {
    // Code for Timer Compare B
}
```
